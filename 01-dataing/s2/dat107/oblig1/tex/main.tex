\section{Introduksjon}


I denne oppgaven skal vi undersøke databasedesign for oss et system som registrer passasje for bompengestasjoner.

Det er oppgitt at systemet skal ta i bruk en database av typen SQL for å registere passasjer, samt krav til hvilke data som skal lagres for hver passering, i tillegg til hvilke  data som skal lagres for hver bil som tar i bruk systemet.

For hver passering skal følgende informasjon lagres:
\begin{itemize}
    \item Hvilken bil som passerte
    \item Hvilken stasjon som ble passert
    \item Tidspunkt for passering
\end{itemize}

For hver bil er det behov for å lagre følgende informasjon:
\begin{itemize}
    \item Registreringsnummer
    \item Eiers navn
    \item Eiers adresse
    \item Eiers e-post
    \item Eiers telefonnummer
\end{itemize}
