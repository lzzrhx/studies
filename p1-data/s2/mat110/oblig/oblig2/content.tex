Newtons metode:
\begin{equation}
\label{newton}
\begin{split}
&\text{1. Gjett at $x_0$ er et nullpunkt.}\\
&\text{2. Gjû¡r fû¡lgende steg om og om igjen:}\\
&x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
\end{split}
\end{equation}

\vspace{0.5cm}

TilnûÎrmet verdi for $\sqrt[3]{2}$ ved bruk av Newtons metode \eqref{newton}:
\begin{align*}
&x = \sqrt[3]{2}&\\
&x^3 = 2\\
&x^3-2 = 0\\
&f(x) = x^3 - 2, \ \ f'(x) = 3x^2\\\\
%&\boxed{x_{n+1} = x_n - \frac{x^3_n - 2}{3x^2_n}}\\\\
&x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\\
&\text{Vi velger } x_0 = 1\\
&x_1 = 1 - \frac{1^3 - 2}{3(1)^2}\\
&x_1 = 1.333333\dots\\\\
&x_2 = 1.33333 - \frac{1.33333^3 - 2}{3(1.33333)^2}\\
&x_2 = 1.263889\\\\
&x_3 = 1.263889 - \frac{1.263889^3 - 2}{3(1.263889)^2}\\
&x_3 = \uuline{1.26003}
\end{align*}

    return x**3 - 2

    x = x - f(x)/Df(x)
    it += 1

LineûÎr tilnûÎrming:
\begin{equation}
\label{linear}
\begin{split}
&\text{Hvis $f(x)$ er deriverbar i $x=a$, da er}\\[0.1cm]
&L(x) = f'(a)(x-a) + f(a)\\[0.1cm]
&\text{en lineûÎr tilnûÎrming til $f(x)$ i $x=a$.}
\end{split}
\end{equation}

\vspace{0.5cm}

TilnûÎrmet verdi for $\sqrt[3]{7,96}$ ved bruk av lineûÎr tilnûÎrming \eqref{linear}:
\begin{align*}
&\text{La } f(x) = \sqrt[3]{x} = x^{\frac{1}{3}}\\
&\text{Da er } f'(x) = \frac{1}{3} x^{- \frac{2}{3}} = \frac{1}{3 \sqrt[3]{x^2}}\\\\
&\text{Det nûÎrmeste tallet til $7.96$ som har en perfekt kubikkrot er $8$ ($\sqrt[3]{8} = 2$).}\\
&\text{Vi lar } x = 8\\
&f(8) = \sqrt[3]{8} = 2\\
&f'(8) = \frac{1}{3 \sqrt[3]{8^2}} = \frac{1}{3 \sqrt[3]{64}} = \frac{1}{3(4)} = \frac{1}{12}\\\\
&\text{Dette setter vi inn i formelen for lineûÎr tilnûÎrming \eqref{linear}:}\\
&a = 8, \ x = 7,96, \ x - a = 7.96 - 8 = -0.04\\
&\sqrt[3]{7,96} \approx 2 + \frac{1}{12} \cdot (-0.04)\\
&\sqrt[3]{7,96} \approx \uuline{1.996667}\\\\
&\text{Er dette overestimat eller underestimat?}\\
%&\text{Fû¡rst finner vi $f''(x)$:}\\
&f''(x) = \frac{1}{3} (- \frac{2}{3} x^{-\frac{5}{3}}) = - \frac{2}{9} x^{-\frac{5}{3}} \ \ \Rightarrow \ \ f''(x) < 0\\
&\text{Vi ser at $f''(x)$ er negativ. Funksjonen er konkav.}\\
&\text{Dette tilsier at at det er et overestimat.}
\end{align*}

Teorem for derivasjon av inverse funksjoner:
\begin{equation}
\label{invers}
&(h^{-1})'(y) = \frac{1}{h'(h^{-1}(y))}
\end{equation}

\vspace{0.5cm}

Finn $(h^{-1})'(13)$ for &h(x) = x^3 + 5$ uten ûË bestemme $h^{-1}(x)$:
\begin{align*}
&h(x) = 13\\
&x^3 + 5 = 13\\
&x^3 = 8\\
&x = 2\\
\Rightarrow \ &h^{-1}(13) = 2\\\\
%&\text{Videre, finner vi } h'(x):\\
&h'(x) =  3x^2\\
&h'(h^{-1}(13)) = h'(2) = 3(2)^2 = 12\\\\
&\text{Vi setter $h'(h^{-1}(13))$ inn i \eqref{invers}:}\\
&(h^{-1})'(13) = \frac{1}{h'(h^{-1}(13))} = \frac{1}{h'(2)} = \frac{1}{12}
\end{align*}
